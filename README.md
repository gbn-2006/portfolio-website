# 🌐 Personal Portfolio Website

A responsive personal portfolio website built using **HTML, CSS, and modern frontend design principles**.
This project highlights my profile, technical skills, projects, and provides a clean interface for interaction.

---

## 🔗 Live Demo

🚀 **View Live Website:**
👉 https://gbn-2006.github.io/portfolio-website/

---

## 🚀 Features

* 📌 Clean and minimal UI design
* 📱 Responsive layout using Flexbox & Grid
* 🧭 Consistent navigation across all pages
* 🧩 Project showcase with structured cards
* 📬 Contact form UI with improved styling
* 🎨 Centralized and reusable CSS styling

---

## 🗂️ Project Structure

```
portfolio-website/
│
├── index.html        # Home page
├── about.html        # About me section
├── projects.html     # Projects showcase
├── gc.html           # Greeting Card project
├── rf.html           # Registration Form project
├── contact.html      # Contact form
├── style.css         # Main stylesheet
└── README.md         # Project documentation
```

---

## 📄 Pages Overview

### 🏠 Home (`index.html`)

* Introduction section with name and tagline
* Navigation buttons to explore the website
* Centered layout using Flexbox

### 👤 About (`about.html`)

* Personal description and background
* Skills and programming languages
* Card-based layout for structured content

### 💼 Projects (`projects.html`)

* Grid-based project display
* Each project includes:

  * Title
  * Technology used
  * Navigation to project files

#### 📁 Project Files

* `gc.html` → Greeting Card project
* `rf.html` → Registration Form project

### 📞 Contact (`contact.html`)

* User-friendly form including:

  * First Name
  * Last Name
  * Email
  * Message
* Styled inputs and buttons for better UX

---

## 🎨 Styling

* All styles are managed in **style.css**
* Techniques used:

  * Flexbox → Layout alignment
  * CSS Grid → Project layout
  * Hover effects & transitions
* Consistent color theme and UI design across pages

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox + Grid)

---

## 📦 How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/gbn-2006/portfolio-website.git
   ```

2. Navigate to the folder:

   ```bash
   cd portfolio-website
   ```

3. Open `index.html` in your browser
